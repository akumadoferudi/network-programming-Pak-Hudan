This is javaSocket readme. Every folder is a Netbeans project folder.

echoServer - simple server that echoing message back to client
echoClient - simple client that send message to server
echoServerSerialization - improvisation of echoServer with object serialization
echoClientSerialization - echoClient with object serialization
messageSerialization - a simple class that implements Serialization in Java. This project should be imported to echoServerSerialization and echoServerSerialization
echoServerMulticlient - echoServer with multiclient support
README.txt - The file you read just now.
